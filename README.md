# Simple C Terminal User Interface
A simple and convenient library for terminal user interface.

## Build & Install
> You may need to use `sudo`
```sh
make all install
```
Or
```
sudo make all install
```

## Use
Pass the `-lsctui` parameter to the C compiler,
And then `#include <sctui/sctui.h>` in your source code.

### skb.h
A simple header only library to handle key press.
See it to use.
